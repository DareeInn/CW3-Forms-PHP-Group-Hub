# CW3-FORMS-PHP-GROUP-HUB

Simple coursework project structure for handling form submissions with PHP.

Files:
- buyagrade.html - form for submitting "payments"
- sucker.php - server-side script to append submissions to suckers.html
- suckers.html - list of submissions
- index.css - styles

Usage:
1. Host the files on a PHP-enabled server (e.g., built-in PHP server: php -S localhost:8000).
2. Open buyagrade.html, submit the form — entries will be added to suckers.html.
